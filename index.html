Javascript is an interpreted, scripting or programming language.Which is most commonly used and it is on client side scripting language because it runs on client's machine inside the web browser.Modern web browsers have their own Javascript engines.After html and css the third layer of standard web technologies is javascript, which enable us to update contents , control multimedia, animate images and almost everything.

As we all know Javascript is single threaded and with this single thread javascript needs to do all of the work on a web page , such as if a user clicks on a button the thread is off doing its thing , what it has been told to do by the user . And if the user immediately click another button it keeps the note in a queue(which is called event queue) that it need to execute that event . And once js finish doing the first task it executes the second event and it keeps on doing the thing until the queue is empty.By this through a loop it goes through the skew of events and executes them and this loop is called event loop.

Session Storage: As the name suggests session storage is only available for the time the browser is running or the tab/window is running and is deleted when the they are closed. Data is temporary here.

Local Storage:Here in local storage if we need the data to be available on an ongoing basis then we need to keep them in here.Data is permanent here until changed or delted.

The data stored in localStorage or sessionStorage can easily be read or changed by the user so we shouldn't rely on any of htem
